This is not a fixed plan, open to changes that make sense to keep this a bit more simple
Create a client portal for students to login.

Students can book tutoring time slots, would need to take payments at time of booking so we dont have students booking times and not showing up.
Priyanka needs to be able to make time slots available for booking (Shes not always available so needs to be able to configure the booking).
Students can choose a subject at time of booking (Pre set subjects) and explain the section that they need help with (Student can write about the issues they are facing)
Added bonus students can upload documents/pictures of the work they want to go over.

Longer term goals:
Publish videos on the portal going through our practice questions. So students can download the questions attempt them, mark them for themselves and access video guides for the areas/sections they need assistance with
We create like a student question forum which is public so students can post questions, Priyanka can respond to the questions and all other students can look through the questions

The longer term goal is to create a subscription based model for students to be a member of the portal and gain access to the forums and video guides.
We know we cant really stop people from downloading guides and practice questions that we post and sharing them with their friends (and we dont really want to stop them doing this) so the idea is to make those kinds of things free to access while having a premium section which students can subscribe too.
